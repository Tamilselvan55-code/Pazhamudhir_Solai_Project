# Tiruchendur Murugan Pazhamudhir Solai - Grocery Delivery Application

Production-ready hyperlocal grocery delivery application built with React, Vite, Node.js, Express, Prisma, and Google Maps API.

## Project Structure
- `frontend/` - React SPA (Vite, TailwindCSS, Zustand, Google Maps)
- `backend/` - Express REST API (Prisma ORM, Socket.IO, Auth)

## Setup & Running

### Backend
```bash
cd backend
npm install
npx prisma generate
npm start
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
